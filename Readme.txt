
___________________Part 2 of Test: Anagram Finder________________________

Abstract: Anagram Finder is app lets you search anagrams from dictionary.txt 
	  using nodeJS Steam. 

Application structure:
          It is server side rendered App.
	  It uses ExpressJS, PugJS.
	  It uses Twitter Bootstrap 4 css framework along with css written in Pug template
	  Testing: JasminJS.

Technology used:
	  ExpressJS, PugJS, JQuery
	  Twitter Bootstrap 4
	  JasminJS  

Please find Part2 zip file and extract it and cd into it 

Requirement: 
NodeJS 

Instruction for execution:

execute following commands in cmd:

step 1) npm install 
step 2) node index.js
step 3) Please visit http://localhost:8080 in browser.
step 4) Search for word and enjoy :)
step 5) execute test by opening new cmd and type npm test
